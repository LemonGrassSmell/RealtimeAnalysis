Quick Start

Download releases, and unzip. Copy the folder to destined cluster, and specify hostname of machines running Loader in loaders file, then run ./sbin/deploy.sh, which will distribute needed files to all loader machines.

Run ./sbin/start-loaders.sh, and all loaders will start in background. Check running file will see running logs.

Run ./sbin/loaderClient-daemon.sh $basefile $topic $speed $recordlen $sf
